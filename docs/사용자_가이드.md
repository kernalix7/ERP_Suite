# ERP Suite 사용자 가이드

## 1. 로그인/로그아웃

### 로그인
1. 브라우저에서 시스템 URL에 접속합니다.
2. 관리자가 발급한 아이디와 비밀번호를 입력합니다.
3. **로그인** 버튼을 클릭합니다.

> **주의:** 비밀번호를 5회 연속 틀리면 1시간 동안 계정이 잠금됩니다. 잠금 해제는 관리자에게 문의하세요.

### 로그아웃
- 좌측 사이드바 하단의 사용자 정보 옆 **로그아웃 아이콘**을 클릭합니다.

## 2. 대시보드

로그인 후 가장 먼저 보이는 화면입니다.

- 주요 지표(매출, 재고 현황, 최근 주문 등)를 한눈에 확인할 수 있습니다.
- 좌측 사이드바를 통해 각 모듈로 이동합니다.
- 사이드바는 상단 햄버거 메뉴 아이콘으로 접기/펼치기가 가능합니다.

## 3. 재고관리

### 제품 등록
1. 사이드바에서 **재고관리 > 제품 관리**를 클릭합니다.
2. 우측 상단 **등록** 버튼을 클릭합니다.
3. 필수 항목을 입력합니다:
   - **제품코드**: 고유한 코드 (예: `FD-001`)
   - **제품명**: 제품 이름
   - **제품유형**: 원자재 / 반제품 / 완제품 중 선택
   - **단위**: 기본 `EA`
   - **판매단가 / 원가**: 금액 입력
   - **안전재고**: 최소 유지 수량
4. **등록** 버튼을 클릭하여 저장합니다.

### 카테고리 관리
- **재고관리 > 카테고리**에서 제품 분류를 관리합니다.
- 상위 카테고리를 지정하여 계층 구조를 만들 수 있습니다.

### 창고 관리
- **재고관리 > 창고 관리**에서 창고를 등록합니다.
- 창고코드, 창고명, 위치를 입력합니다.

### 입출고 등록
1. **재고관리 > 입출고**에서 **등록** 버튼을 클릭합니다.
2. 입출고 유형을 선택합니다:
   - **입고**: 외부에서 물품 입고
   - **출고**: 외부로 물품 출고
   - **재고조정(+/-)**: 실사 후 재고 보정
   - **생산입고/출고**: 생산 시 자동 생성
   - **반품**: 반품 처리
3. 제품, 창고, 수량, 날짜를 입력하고 저장합니다.

> 입출고 등록 시 해당 제품의 현재고가 자동으로 반영됩니다.

### 창고간 이동
1. **재고관리 > 창고간이동**에서 **등록** 버튼을 클릭합니다.
2. 출발창고, 도착창고, 제품, 수량, 이동일을 입력합니다.
3. 저장하면 출발창고에서 출고, 도착창고에 입고가 자동 처리됩니다.

### 재고현황
- **재고관리 > 재고현황**에서 전체 제품의 현재고와 안전재고 대비 상태를 확인합니다.
- 안전재고 미달 제품은 빨간색으로 표시됩니다.

## 4. 생산관리

### BOM(자재명세서) 등록
1. **생산관리 > BOM 관리**에서 **등록** 버튼을 클릭합니다.
2. 완제품과 해당 완제품을 만들기 위한 원자재/반제품 목록을 등록합니다.
3. 각 자재의 필요 수량을 입력합니다.

### 생산계획 수립
1. **생산관리 > 생산계획**에서 새 계획을 등록합니다.
2. 생산할 제품, 계획 수량, 계획일을 입력합니다.

### 작업지시
1. **생산관리 > 작업지시**에서 생산계획에 따른 작업지시를 생성합니다.
2. 작업지시 상태를 관리합니다 (대기 → 진행중 → 완료).

### 생산실적
1. **생산관리 > 생산실적**에서 실제 생산 결과를 기록합니다.
2. 생산 완료 시:
   - 완제품이 자동으로 입고 처리됩니다.
   - BOM에 따라 원자재가 자동으로 출고 처리됩니다.

## 5. 판매관리

### 거래처 등록
1. **판매관리 > 거래처**에서 거래처 정보를 등록합니다.
2. 사업자번호, 대표자명, 연락처 등을 입력합니다.

### 고객 등록
- **판매관리 > 고객**에서 개인 고객 정보를 관리합니다.

### 주문 생성
1. **판매관리 > 주문**에서 **등록** 버튼을 클릭합니다.
2. 고객/거래처, 주문 제품, 수량을 입력합니다.
3. 부가세(10%)가 자동 계산됩니다.
4. 주문 확정 시 출고가 자동으로 연동됩니다.

### 수수료 관리
- **판매관리 > 수수료율**에서 파트너별 수수료율을 설정합니다.
- **판매관리 > 수수료내역**에서 발생한 수수료를 확인합니다.
- **판매관리 > 정산요약**에서 파트너별 정산 현황을 조회합니다.

## 6. AS관리

### AS 접수
1. **AS관리 > AS 접수**에서 **등록** 버튼을 클릭합니다.
2. 고객 정보, 제품, 증상을 입력합니다.
3. 보증기간 내 여부가 자동으로 표시됩니다.

### 수리 처리
1. 접수된 AS 건을 클릭하여 상세 페이지로 이동합니다.
2. 수리 내용을 기록하고 상태를 변경합니다 (접수 → 수리중 → 완료).
3. 수리이력이 자동으로 추적됩니다.

## 7. 회계관리

### 세금계산서
- **회계관리 > 세금계산서**에서 매출/매입 세금계산서를 등록합니다.
- 공급가액과 세액이 자동 계산됩니다.

### 부가세 집계
- **회계관리 > 부가세 집계**에서 분기별 부가세 현황을 조회합니다.

### 고정비 관리
- **회계관리 > 고정비 관리**에서 임대료, 인건비, 장비 리스비 등 고정 지출을 등록합니다.

### 손익분기점
- **회계관리 > 손익분기점**에서 Chart.js 기반 차트로 손익분기점을 분석합니다.

### 월별 손익
- **회계관리 > 월별 손익**에서 월별 손익계산서를 확인합니다.

### 전표 작성
1. **회계관리 > 전표**에서 **등록** 버튼을 클릭합니다.
2. 전표 유형(입금/출금/대체)을 선택합니다.
3. 계정과목, 금액, 적요를 입력합니다.

### 원천징수
- **회계관리 > 원천징수**에서 원천징수 내역을 관리합니다.

## 8. 투자관리

### 투자자 등록
- **투자관리 > 투자자 관리**에서 투자자 정보를 등록합니다.

### 투자 라운드
- **투자관리 > 투자 라운드**에서 투자 라운드를 생성하고 투자 내역을 기록합니다.

### 지분 현황
- **투자관리 > 지분 현황**에서 도넛차트로 지분율을 확인합니다.

### 배당/분배
- **투자관리 > 배당/분배**에서 배당금 및 수익분배를 기록합니다.

## 9. 스마트스토어 연동

### 설정
1. **스마트스토어 > 스토어 설정**에서 네이버 API 인증 정보를 입력합니다.

### 주문 동기화
- **스마트스토어 > 스토어 대시보드**에서 동기화 버튼으로 주문을 가져옵니다.
- **스마트스토어 > 스토어 주문**에서 동기화된 주문 목록을 확인합니다.
- **스마트스토어 > 동기화 이력**에서 동기화 수행 기록을 조회합니다.

## 10. 문의관리

### 문의 등록/관리
1. **문의관리 > 문의 목록**에서 채널별(스마트스토어/인스타/카카오) 문의를 등록합니다.
2. 문의 상세에서 **AI 답변 생성** 버튼을 클릭하면 Claude AI가 답변 초안을 생성합니다.
3. 생성된 초안을 검토/수정 후 답변을 등록합니다.

### 답변 템플릿
- **문의관리 > 답변 템플릿**에서 자주 사용하는 답변 양식을 등록합니다.

## 11. 정품등록

- **정품등록** 메뉴에서 시리얼번호 기반 정품 인증을 관리합니다.
- 보증 시작일, 만료일이 자동으로 계산됩니다.
- 시리얼번호로 정품 여부를 조회할 수 있습니다.

## 12. Excel 다운로드

각 목록 페이지에서 Excel 다운로드 기능을 사용할 수 있습니다.

1. 해당 목록 페이지로 이동합니다 (예: 제품 관리, 주문 목록).
2. **Excel 다운로드** 버튼을 클릭합니다.
3. 스타일링된 `.xlsx` 파일이 다운로드됩니다.

## 13. 백업/복원

> 관리자(admin) 권한이 필요합니다.

### 백업
1. 사이드바에서 **시스템관리 > 백업/복원**을 클릭합니다.
2. **백업 생성** 버튼을 클릭합니다.
3. 생성된 백업 파일을 다운로드합니다.

### 복원
1. **백업/복원** 페이지에서 백업 파일을 업로드합니다.
2. **복원** 버튼을 클릭합니다.

> **주의:** 복원 시 현재 데이터가 백업 시점의 데이터로 대체됩니다. 복원 전 반드시 현재 데이터를 백업해 두세요.
